## Packages
Needing a Golang eco system and some basics
````
pkg_add go-- packer-- git--
````

## Repo
````
git clone https://github.com/double-p/packer-builder-openbsd-vmm.git
````

## builds
Set ````GOPATH```` (default: ~/go), if the 1.4GB dependencies wont fit.
````
make
make install
````
